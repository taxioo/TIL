## 코드 블럭(code block)

backquote(backtic) 3개 + Enter로 코드블럭 생성

```python
a = 1
b = 2
c = a + b
print(c)

def my _func(n)
	return n+1

```



## 표(table)

- 우클릭해서 삽입 --> 표
- pipe(l)로 열을 구분하고 Enter로 표 생성
- 행 추가는 ctrl+enter로 행 추가

| 명령어  | 설명      | 사용 예시      |
| ------- | --------- | -------------- |
| `mkdir` | 폴더 생성 | `mkdir aaa`    |
| `cd`    | 폴더 이동 | `cd aaa`       |
| `touch` | 파일 생성 | `touch a. mid` |
